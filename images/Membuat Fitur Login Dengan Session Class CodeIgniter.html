Membuat Fitur Login Dengan Session Class CodeIgniter

08/08/2014    Reza Indra    33917     Website

Hai, dalam tutorial kali ini kita akan membuat fitur login dengan Framework CodeIgniter. Untuk membuat fitur login, pasti membutuhkan sebuah fasilitas untuk mengenali id seorang user, untuk itu CodeIgniter sudah mempersiapkan class Session yang nanti akan digunakan dalam membuat sistem login ini.

Langkah pertama, buatlah file login.php dalam folder Controller teman-teman.

Jalankan text editor PHP Designer, tulis baris program berikut. Di dalam class Login buatlah beberapa function.

Kemudian buat juga folder login dan di dalamnya buat file index.php dan admin_page.php.

Sekarang ketiklah baris program berikut pada function __construct() untuk load library session.

Ketik baris program berikut pada function index().

Sekarang, buka file index.php yang ada di dalam folder login. Buat form untuk login.

Nah, karena jika berhasil login akan di-redirect ke halaman admin, berarti kita buat funtion admin_page() untuk me-load pada halaman admin_page.

Buka file admin_page.php yang ada dalam folder login dan cetak session dan buat link logout-nya.

Sekarang kita coba jalankan http://codeigniter/index.php/login.

Jika username dan password benar.

Buat funtion logout() untuk menghapus session dan redirect kembali ke halaman login.
